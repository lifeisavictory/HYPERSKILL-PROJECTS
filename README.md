## Co je Hyperskill?

Hyperskill je vzdělávací platforma zaměřená na výuku programování a vývoje softwaru
prostřednictvím projektově orientovaných kurzů. Platforma je vyvíjena společností
**JetBrains** — tvůrci IntelliJ IDEA a dalších vývojářských nástrojů.

**Klíčové vlastnosti platformy:**
- 300+ praktických projektů (Java, Python, Kotlin, Spring Boot, SQL a další)
- Poměr výuky: **80 % praxe, 20 % teorie**
- Integrace s JetBrains IDE (IntelliJ IDEA)
- Strukturované kurzy s certifikátem po dokončení
- Více než 60 kurzů pokrývajících moderní technologie

## Moje studijní zaměření

Aktuálně studuji kurzy zaměřené na:
- ☕ **Java** — objektově orientované programování, kolekce, streamy
- 🌱 **Spring Boot** — tvorba REST API a backend aplikací
